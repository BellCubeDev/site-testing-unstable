Nexus Stats Scraper

Nexus Mods stores statistics for all mods publicly, down to the day. However, there is no easy way to get the raw data for these statistics—at least, not without this tool. This tool will extract the raw data from the Nexus Stats page and output it in multiple formats, such as JSON and CSV. Additionally, it will also (eventually) display the File Update Tree, which visually shows the history of file updates for a mod as your mod manager sees it.
